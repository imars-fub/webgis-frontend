Contrast enhanced colour image mosaic from multiple orthoimage strips of the High-Resolution Stereo Camera (HRSC) of ESA's Mars Express produced by differential rectification using an HRSC DTM of this area

Product type: image mosaic (Level-5)

Grid resolution: 50 m per pixel

Note: currently, only the MC11-E quadrangle is available. The western part of MC11 is currently being assembled.

Data source: - currently not available for public download.
