Contrast enhanced image mosaic from multiple orthoimage strips of the Context Camera (CTX) of NASA's Mars Reconnaissance Orbiter (MRO)

The images have been produced by differential rectification using the HRSC image mosaic for MC-11

Product type: image mosaic (Level-5)

Grid resolution: 12.5 m per pixel

Note: currently, only the MC11-E quadrangle is available.

Data source: - currently not available for public download.

Reference to the single-strip processing:
http://www.hou.usra.edu/meetings/lpsc2016/pdf/2034.pdf

Reference to the mosaic processing:
Michael et al. (2016), Systematic processing of Mars Express HRSC panchromatic and colour image mosaics: Image equalisation using an external brightness reference, doi: 10.1016/j.pss.2015.12.002
